CommunityPulse User Guide

Welcome to the CoomunityPulse user guide. Click Next to learn about the features of CommunityPulse. Click Close to exit the user guide.

Selecting a Proposal

Click on a proposal to explore the related comments. The red box shows a selected proposal. Once selected, the proposal name will turn blue. Click on the proposal again, to unselect it.

Selecting a Topic

Click on a topic of a proposal to explore comments related to that topic only. The red box shows a selected topic. Once selected, the topic name will turn blue. Click on the proposal again, to unselect it.

Sorting by Emotion

Click on the emoticons to sort the proposals by emotion. You can Click for (descending ), (ascending ) and (no ) sort. The red box shows sorting by emotion - Excited

Filtering by Emotion

Click on the emotion bars of a proposal to explore comments related to that emotion only. The red box shows a proposal's comments filtered by the emotion - Happy.

Revision

The emotion for each comment is identified using machine learning. If you do not agree with it, you can revise the emotion category. Hover on the emoticon below the comment to bring up the revision panel. Then click on the emotion you think is correct. The red box shows the emotion below the comment. The green box shows the comment is revised from Excited to Concerned.

Compare

You can select up to two proposals for a side-by side comparison. Click on checkboxes to select proposals. Then Click on the Compare button in the navigation bar. The red box shows selecting proposals and the green box shows the Compare button.

User Guide

This is the end of our user guide. If you want to see this guide again, click on the User Guide button in the navigation bar. The red box shows the User Guide button.
